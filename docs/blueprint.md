# **App Name**: SustentaAI

## Core Features:

- Image Analysis for Sustainability: Analyze uploaded product images (or names) to determine environmental impact using an AI tool and external APIs.
- Environmental Impact Visualization: Display the analyzed product's carbon footprint, water footprint, and overall environmental impact score in an intuitive format, including impact dimension indicators.
- Sustainable Alternatives: Recommend related, more sustainable products with links to purchase, encouraging eco-friendly choices.
- SDG Education: Provide educational content on the UN's Sustainable Development Goals (SDGs) with interactive cards and visual representations.
- Green Map Integration: Integrate a map displaying local green areas and initiatives, promoting engagement with local sustainability efforts.
- Product Data Storage: Store and retrieve product information, including environmental impact data, using Firebase Firestore.
- Database Search: If user types in a product name rather than submitting an image, use that name to attempt lookup within our product database. Show a 'Product not recognized' error message if lookup fails.

## Style Guidelines:

- Primary color: Light green (#A7D98B) to evoke sustainability and freshness.
- Background color: Very light green (#F0FAF3) to provide a clean and airy feel.
- Accent color: Light blue (#8BD0D9) to complement the green tones and add a sense of trust and reliability.
- Body and headline font: 'Inter' sans-serif for a modern, clean, readable design.
- Minimalist icons from Lucide or Heroicons to ensure clarity and simplicity.
- Clean, modern layout with a focus on whitespace and readability. Cards should be used to organize content into logical sections.
- Subtle and dynamic animations to enhance user experience, with elegant transitions and visual feedback.
